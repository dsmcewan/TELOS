# Award-winning UI/UX references — the standard Narcissus reaches for

Concrete, current best-in-class precedent (Awwwards Site-of-the-Year tier). Award-winning is the DEFAULT bar,
not an aspiration. Specifics on *why* each wins — not vibes.

## Reference sites (SOTY / SOTM tier, 2025–2026)
- **Messenger** (Awwwards Site of the Year 2025) — a tiny **WebGL planet** brought to life in-browser. *Why it
  wins:* it turns interaction into **play** — the visitor is a *participant in a living scene*, not a passive
  scroller; GPU-powered physics/lighting/animation that **loads and runs in seconds**. Directly relevant:
  Narcissus's visitor should *participate in* the dev-story, not read it.
- **Lando Norris official site** (OFF+BRAND — Site of the Year 2025) — bold, brand-driven, cinematic.
- **Bruno Simon portfolio 2025** — a browser-based **3D drivable world**; the canonical proof that a fully 3D,
  explorable surface can be both spectacle and navigable.

## Patterns that recur in winners (2026)
- **WebGL / 3D browser experiences** as the emotional core (not a background gimmick).
- **Immersive storytelling** — a narrative you move *through*, spatially.
- **Bold, typography-driven layouts** — type as a primary design element.
- **AI-integrated interfaces**; **spatialized audio**; **purpose-driven UX**.
- **Extreme performance** — Core Web Vitals treated as a hard constraint, not an afterthought (winners *load
  and dazzle in seconds*). This is the counterweight to "over the top": spectacle that is also fast.

## Awwwards evaluation criteria (the scoring rubric to design against)
**Design · Usability · Creativity · Content.** Note Usability sits beside Creativity — award-tier is NOT
beauty at the cost of function, which aligns exactly with Narcissus's two-blade gate (aesthetic AND
deterministic-functional).

## Implication for the flagship
The bar is: a WebGL/3D **participatory** surface (visitor acts, not just scrolls), bold typographic narrative,
immersive story-you-move-through, that **loads and runs in seconds** and scores on Usability as well as
Creativity. The breakout's "Loom on Trial" synthesis targets exactly this (see `breakout-synthesis.md`).
