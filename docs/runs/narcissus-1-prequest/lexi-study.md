# LEXI study — aesthetic reference for Narcissus

**Repo:** `github.com/dsmcewan/LEXI` (confirmed public; fetched 2026-07). LEXI is one of The Eye's own systems —
the design anchor Narcissus's reflection loop targets.

## What LEXI is
A **multi-agent forensic NLP platform**: four LLMs behind a complexity-routing pipeline, bias validation held
separate from generation, cross-model consensus checks. It reconstructs event narratives from large,
multi-year unstructured communication datasets, explicitly distinguishing **evidence from uncertainty**
(behavioral pattern recognition, automated timelines, a source-grounded chat assistant). This ethos —
evidence-vs-uncertainty, cross-model consensus, forensic rigor — is deeply on-brand for a TELOS surface.

## Front-end / design language (as documented + inferable from the stack)
- **Stack:** React (Vite) + **Tailwind CSS** + vanilla CSS; **Three.js / WebGL** for hero visuals;
  **D3.js** for timeline data-viz.
- **Signature hero:** a **Three.js WebGL "Ferrofluid Orb"** on a standalone landing dashboard — a dark,
  living, fluid-metal centerpiece. (This is the direct ancestor of the breakout's "ferrofluid weave" idea.)
- **Data-viz forward:** an interactive **D3 event-timeline** (`event-timeline.html`) — the story-as-timeline
  pattern.
- **Aesthetic (inferred):** dark, sophisticated, forensic-dashboard; WebGL hero as the emotional anchor;
  data-visualization as first-class content, not decoration.
