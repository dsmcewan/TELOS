## Headline findings from the breakout
- **Convergence (unprompted):** all four seats reached a **loom/weave** metaphor — the Clotho weave as literal
  woven thread (myth-faithful to Clotho the spinner).
- **Shared fatal risk (unanimous):** the **functional E2E blade** — physics/free-form WebGL interactions are
  non-deterministic and can't pass a deterministic E2E test.
- **Unanimous resolution:** **"WebGL is paint, never truth"** — canvas is a pure view-layer; all state/action
  in a deterministic **XState machine + typed command registry**; every 3D element shadowed by an invisible
  semantic HTML control (→ testable AND accessible); `?e2e=1` freezes RNG/physics for deterministic tests.
- **Recommended direction:** synthesis **"The Loom on Trial / Crucible Weave"** — award-tier suspended-loom
  spectacle, but the visitor moves among discrete forensic **stations** (bounded, testable) with an Evidence
  Ledger grounding every claim in the real institutional-memory record, and a reverse-topological "unraveling"
  climax.
