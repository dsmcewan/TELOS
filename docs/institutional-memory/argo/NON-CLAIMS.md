# Argo — non-claims

- **No autonomous Argo runner exists.** Argo names a role and protocol; every slice
  is human-initiated. Do not build, assume, or reference an Argo service.
- **A passing gate artifact is not implementation authority.** It is the
  precondition. The Eye grants authority; formal wiring of the artifact into TELOS
  authority is a recorded open question.
- **The comprehension gate is open-book.** It proves correct authority resolution
  and record reading — not deep understanding. Expected answers are stored in-band
  by design (determinism over model-graded judgment). Do not oversell a pass.
- **Acceptance is not a zero-defect claim.** Slice 4a was accepted WITH a deferred
  backlog and a bounded lone dissent; the backlog is part of the record.
- **Argo does not choose scope.** Plan ambiguities escalate (Eye → Daedalus →
  TELOS); they are never designed around in code.
