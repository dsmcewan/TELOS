01 — Builder ≠ Certifier

The system that distrusts itself

TELOS is a build-gate where no seat's self-report is trusted. Independent AI seats produce signed packets; a deterministic gate certifies merge-readiness from disk, signatures, and provenance — never from what a model claims. The system was built to be its own harshest critic.
